Nos mostraría el valor 5 ya que cuando ejecutamos setTimeout el ciclo for ha terminado y el valor actual que tiene i es 5.

Lo solucionaria con el siguiente código: var i = 0; setInterval(function () { if(i < 5){ console.log(i); i++; } }, 1000)
